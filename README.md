# Tic-Tac-Toe

A premium, dark-themed Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features
- **User vs Computer**: Play against a basic AI.
- **Premium Design**: Glassmorphism, gradients, and smooth animations.
- **Winning Line Animation**: Visual indicator for the winning combination.
- **Auto-Reset**: Game automatically resets 1 second after a win or draw.
- **Score Tracking**: Keeps track of wins and draws.

## Live Demo
[tic-tac-toe](https://rz-tictactoe.netlify.app/)

## How to Play
1. Click on any empty cell to place your 'X'.
2. The computer will automatically play 'O'.
3. The first to get 3 in a row wins!
